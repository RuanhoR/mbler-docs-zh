## 使用

### 前置

- Git
- nodejs

### 下载

```bash
git clone https://github.com/RuanhoR/mbler-docs.git && cd mbler-docs
```

### 装依赖

```bash
npm i -g vitepress
pnpm install
```

### 启动

```bash
npm run docs:dev
```
