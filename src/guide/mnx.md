# MNX 市场

MNX 是一个构建在 MCX 生态之上的 Minecraft Bedrock 附加包市场。它允许开发者通过中心化的注册表发布、分发和管理他们的附加包。

## 概述

- **公共注册表**：浏览和下载社区附加包
- **版本管理**：使用语义化版本控制和合适的发布渠道
- **基于作用域的发布**：在用户/团队命名空间下组织
- **令牌认证**：为自动化发布提供安全的 API 访问

## 如何发布

1. 使用 `mbler build` 构建您的附加包
2. 发布到 MNX：`mbler publish`
3. 通过 MNX 网页界面管理您的包
