# mbler

## 概述

mbler 是一个 Minecraft Bedrock 附加包开发工具。它基于 Node.js 和 TypeScript，可在 Windows、Linux、macOS 和 Android（Termux）上运行。

- [GitHub 仓库](https://github.com/RuanhoR/mbler)
- 文档（就是当前页面）

## 为什么选择 mbler？

- **解决 SAPI 版本混乱** — 自动解析目标 Minecraft 版本对应的 `@minecraft/server` 依赖
- **最简单的附加包工作流** — 一条命令即可构建、监视和打包
- **TypeScript 支持** — 使用 TypeScript 编写行为包脚本，自动打包
- **国际化 (i18n)** — CLI 支持英文和中文
- **MCX DSL** — 类 Vue 的 DSL，替代冗长的 JSON 配置，用于定义物品、实体、方块、UI 和事件。[了解更多](/guide/mcx)
