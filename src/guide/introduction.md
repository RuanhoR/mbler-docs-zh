# mbler
## 概述
mbler 是一个开发工具。它基于 nodejs 和 typescript。你可以在 Windows、Android（安装 Termux 层）、Linux、Mac 等操作系统上使用它。

关于
- [github 仓库](https://github.com/RuanhoR/mbler)
- 文档（就是当前页面）

## 我们为什么要做它
- 它可以解决 Minecraft SAPI 版本混乱问题
- 它可以用最简单的方法开发 Minecraft 插件
- 它可以用 TypeScript 开发 Minecraft 插件
- 它支持国际化 (i18n)
- 你不需要阅读官方 JSON 教程，只需编写 MCX（非常非常难读）。(这是关于 mbler 的 [mcx](/guide/mcx)，还处于测试阶段)
